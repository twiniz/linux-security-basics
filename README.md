## Key Linux Commands Practiced

| Command | Purpose |
|-------|--------|
| `ls` | List files and directories |
| `pwd` | Show current directory |
| `cd` | Navigate directories |
| `cat` | View file contents |
| `chmod` | Change file permissions |
| `chown` | Change file ownership |
| `whoami` | Display current user |
| `history` | View command history |

## Security Relevance

Understanding Linux is essential for cybersecurity because:
- Misconfigured permissions can lead to privilege escalation
- Logs help detect suspicious activity
- User management impacts access control
- Many servers and security tools run on Linux
